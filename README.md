# 🧮 Python CLI Calculator

A command-line calculator built with Python that performs basic mathematical operations while handling invalid input and common runtime errors.

---

## Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🔼 Power
- 🧮 Modulus
- ✅ Input validation
- ⚠️ Error handling for division/modulus by zero
- 🔁 Continuous calculations until the user chooses to quit

---

## Technologies Used

- Python 3

---

## How to Run

Clone the repository:

```bash
git clone https://github.com/TheAbdullahSaleem/python-cli-calculator.git
```

Run the program:

```bash
python main.py
```
